Driver Drowsiness Detection using CNN-KAN and LSTM-KAN Stacking

Developed an advanced driver drowsiness detection system using a novel stacking approach that combines Convolutional Neural Networks (CNN) with Kolmogorov-Arnold Networks (KAN) and Long Short-Term Memory (LSTM) with KAN. This innovative approach enhances the accuracy and reliability of drowsiness detection in real-time driving scenarios.

Key Features:

  • Hybrid architecture combining CNN-KAN and LSTM-KAN models
  • Real-time facial feature extraction and analysis
  • Temporal sequence processing for improved accuracy
  • Knowledge-augmented learning for enhanced pattern recognition
  • Multi-state classification (Drowsy/Awake)
  • Robust performance across different lighting conditions
